# Mood Memes - Multiplayer 
A fun, interactive, and serverless multiplayer web game where you and your friends can play using your own phones or computers! Built entirely with HTML, CSS, and Vanilla JavaScript, it uses the **PeerJS / WebRTC** infrastructure for seamless Peer-to-Peer connections.

## 🚀 Features (ENG):
* **Multiplayer:** Anyone can instantly join via their browser.
* **Room Creation & Joining:** The host creates a room, and players connect instantly using the generated Room Code.
* **Fully Customizable:** The host can upload custom photos/memes from their device gallery directly into the game pool.
* **Sweet Neon / Glassmorphism UI:** A sleek, modern, and eye-friendly pinkish pastel design.
* **Smart Card Dealing:** A unique deck logic that prevents dealing the exact same card in a single hand.
* **100+ Built-in Scenarios:** Includes over 100 funny and relatable "Situation/Scenario" sentences.

## 🎮 How to Play
1. **The Host (Admin)** starts the game by clicking "Create Room" and shares the 5-digit **Room Code** with friends.
2. The Host can upload new meme photos to the pool before or during the game.
3. **Players** click "Join Room," enter their nickname and the Room Code to enter the lobby.
4. When the game starts, a random **Scenario Sentence** appears on the screen (e.g., *"Me waking up on a Monday morning"*).
5. Players select the funniest photo/meme from their dealt cards (hand of 5) that best reacts to the situation.
6. Once all selected cards are dropped to the table, enjoy the funny combinations and laughs!
7. The Host clicks "Next Round" to deal new cards and continue.

# Mood Oyunu - Çok Oyunculu 🇹🇷 (TUR):

Arkadaş grubunuzla veya ailenizle kendi cihazlarınızdan bağlanarak oynayabileceğiniz, eğlenceli ve etkileşimli bir web tarayıcı oyunu! Tamamen HTML, CSS ve JavaScript (Vanilla) kullanılarak geliştirilmiş olup, sunucusuz (serverless) eşler arası (Peer-to-Peer) bağlantı sağlayan **PeerJS / WebRTC** altyapısıyla çalışmaktadır.

## 🚀 Özellikler (TUR)
* **Çok Oyunculu:** Herkes kendi telefonundan veya bilgisayarından anında tarayıcı üzerinden katılabilir.
* **Oda Kurma ve Katılma:** Oyun yöneticisi bir oda kurar ve oluşturulan kod ile diğer oyuncular anında odaya bağlanır.
* **Tamamen Kişiselleştirilebilir:** Oyun yöneticisi cihazından (galerisinden) istediği kadar fotoğrafı "Oyun Kartı / Caps" havuzuna ekleyebilir.
* **Tatlı Neon / Glassmorphism Tasarım:** Göz yormayan, pembe/pastel modern ve şık arayüz (UI).
* **Akıllı Kart Dağıtımı:** Aynı elde tamamen aynı iki kartın gelmesini engelleyen özel kart destesi mantığı.
* **100+ Hazır Senaryo:** Oyuna entegre edilmiş, komik ve günlük yaşantıdan 100'den fazla "Durum/Senaryo" cümlesi bulunur.

## 🎮 Nasıl Oynanır?
1. **Kurucu (Yönetici)** "Oda Kur" diyerek oyunu başlatır. Kendisine verilen 5 haneli **Oda Kodunu** arkadaşlarına söyler.
2. Yönetici isterse oyuna başlamadan veya oyun esnasında galerisinden yepyeni komik fotoğrafları havuza yükleyebilir.
3. **Oyuncular**, ana ekrandan "Odaya Katıl" diyerek kendi nicklerini ve yöneticinin verdiği oda kodunu girerek lobiye giriş yaparlar.
4. Oyun başladığında ekranda rastgele bir **Senaryo Cümlesi** (Örn: *"Pazartesi sabahı uyanınca ben"*) belirir.
5. Herkesin ekranında 5 adet fotoğraf/caps bulunur. Oyuncular duruma **en komik uyan** fotoğrafı seçip masaya atarlar.
6. Tüm fotoğraflar masaya düştüğünde ortaya çıkan komik kombinasyonlarla kahkaha atılır!
7. Yönetici "Sonraki Duruma Geç" diyerek yeni tura geçer ve baştan yeni kartlar dağıtılır.

## 💻 Tech Stack / Teknolojiler
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Network:** PeerJS (WebRTC)
* **UI/UX:** Light Glassmorphism, CSS Variables
